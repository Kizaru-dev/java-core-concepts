# Java Concepts Practice ☕

This repository contains my practice and revision of core Java concepts.
It includes small programs, examples, and exercises to strengthen my understanding of Java fundamentals.

## 📚 Topics Covered

* Java Basics (variables, data types, operators)
* Control Flow (if-else, switch, loops)
* Methods
* Object-Oriented Programming (OOP)

    * Encapsulation
    * Inheritance
    * Polymorphism
    * Abstraction
* Arrays
* Strings
* Exception Handling
* Collections Framework
* Multithreading
* File Handling

## 📂 Project Structure

## 🎯 Purpose

* Revise core Java concepts
* Practice writing clean Java code
* Prepare for coding interviews
* Maintain a structured record of learning

## 🚀 How to Run

1. Clone the repository
2. Open the project in any Java IDE (IntelliJ / Eclipse / VS Code)
3. Compile and run the Java files.
